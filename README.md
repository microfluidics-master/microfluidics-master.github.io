# Microfluidics Master — Jekyll Site

This repository hosts the GitHub Pages site for the **Master in Microfluidics & Miniaturization**.

## Quick start

1. In this repo settings, go to **Pages** → Source: *Deploy from a branch*, Branch: `main`, Folder: `/(root)`.
2. Edit `_config.yml` to set your `url` (and `baseurl` if this is a project site).
3. Update pages in `index.md`, `presentation.md`, `degree-information.md`, `contact.md`.
4. Replace the favicons in `assets/img/`.

Built with the Jekyll **minima** theme and plugins: `jekyll-feed`, `jekyll-seo-tag`, `jekyll-sitemap`.
